## System Overview

The Gene Keys is a contemplative wisdom system created by Richard Rudd, synthesizing:
- **I Ching** (64 hexagrams as archetypal states)
- **Human Design** (gate positions and activations)
- **Genetics** (64 codons, 20 amino acids)
- **Contemplation Practice** (Shadow → Gift → Siddhi transformation)

### Core Philosophy

The Gene Keys proposes that each of the 64 genetic codons corresponds to a specific archetypal frequency that can be experienced at three levels:

1. **Shadow** (Fear-based consciousness): The contracted, reactive state
2. **Gift** (Love-based consciousness): The balanced, creative state
3. **Siddhi** (Unity consciousness): The transcendent, enlightened state

> "The Gene Keys are not about changing who you are—they are about uncovering who you've always been."

### Key Principles

1. **Contemplation over Analysis**: Gene Keys are meant to be lived with, not merely studied
2. **No Hierarchy**: All 64 keys are equal; none is "better" than another
3. **Spectrum of Consciousness**: Shadow-Gift-Siddhi is a frequency spectrum, not stages
4. **Personal Activation**: Your birth chart activates specific keys for your journey
5. **Collective Field**: The 21 Codon Rings connect individual keys to collective themes

### Line Derivation

The line (1-6) within a Gene Key is derived from the degree within that key's zodiac span:

```
Gene Key span: 5.625°
Each line: 5.625° ÷ 6 = 0.9375°

Example: Sun at 15° Aries
- Gene Key 21 spans 11.25° - 16.875° Aries
- 15° - 11.25° = 3.75° into the key
- 3.75° ÷ 0.9375° = Line 4
```

### Zodiac Calculation

For accurate Gene Key activation:

```typescript
function getGeneKeyForDegree(absoluteDegree: number): { key: number; line: number } {
  // 64 keys across 360 degrees
  const keySpan = 360 / 64;  // 5.625°
  const lineSpan = keySpan / 6;  // 0.9375°

  // Gene Keys start at a specific offset (not 0° Aries)
  // The actual starting point needs verification from source material
  const adjustedDegree = (absoluteDegree + offset) % 360;

  const keyIndex = Math.floor(adjustedDegree / keySpan);
  const degreeWithinKey = adjustedDegree % keySpan;
  const line = Math.floor(degreeWithinKey / lineSpan) + 1;

  return { key: keyIndex + 1, line: Math.min(line, 6) };
}
```
